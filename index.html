Automatización de despliegues multi-lenguaje en Kubernetes

$ gitlab-ci pipeline deploy --multi-lang

Mediante pipelines de CI/CD de GitLab, implementé un sistema automatizado de despliegue de proyectos sobre Kubernetes. Desde un frontend, los usuarios subían código en múltiples lenguajes (Java, React, Python, Node.js, Vue, .NET, entre otros) y el pipeline ejecutaba automáticamente: análisis de seguridad y vulnerabilidades, containerización de la aplicación, despliegue en cluster de Kubernetes, creación de Ingress y registro de subdominio en Cloudflare. Sistema diseñado para desplegar code challenges en una plataforma dedicada a evaluaciones técnicas.

📍 Empresa: Waimatech

Creación de Infraestructura desde cero para fintech

$ terraform apply -var-file="fintech-prod.tfvars"

Utilizando Terraform, desplegué desde cero una infraestructura completa en Azure para un sistema de gestión de financieras online. El proyecto incluyó: clusters de Kubernetes, servidores de bases de datos, bastions, servidores FTP, blob storages, despliegue y configuración de WSO2 API Manager, pipelines en GitLab para despliegue de APIs, configuración de monitorización (Prometheus, Loki, Pingdom) y servicio de mailing con Amazon SES. Infraestructura productiva completa y escalable para operaciones financieras.

📍 Empresa: CloudAPPi

Experiencia práctica con control plane de Kubernetes y gestión de infraestructura multi-cloud.
